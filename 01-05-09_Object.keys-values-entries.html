🌟 핵심 요약
    keys → 이름표 모음
    values → 값 모음
    entries → 이름표+값 세트

    👉 쉽게 말하면 문구점 장부 같아:
    keys = 물건 이름만 적힌 종이
    values = 가격만 적힌 종이
    entries = 이름+가격이 같이 적힌 종이
🍎일반 객체(Map,Set,Array)에서 키,값,전체 추출하여 배열에 넣는것 어떻게 씀? 
    Object.keys(obj) – 객체의 키만 담은 배열을 반환 
    Object.values(obj) – 객체의 값만 담은 배열을 반환 
    Object.entries(obj) – [키, 값] 쌍을 담은 배열을 반환 

    <>맵에선 obj.keys( ) 이렇게 했었음
    <>맵에선 obj.values( ) 이렇게 했었음
    <>맵에선 obj.entries( ) 이렇게 했었음
객체에 .map .filter와 같은 배열 전용 메서드를 사용할 수 있을까요? 
NO ! 
        그러나, Object.entries(객체) 랑 Object.fromEntries(객체) 와 같은 전용 메서드를 사용하여 사용 가능
        메서드 사용할 수 있다.
