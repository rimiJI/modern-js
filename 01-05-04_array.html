배열 Array
배열의 본질은 객체이다.

위의 것 선언방법 두가지

let arr=new Array();
let arr=[]

배열은 인덱스넘버가지고 수정 가능?

YES
let fruits = ["사과", "배", "포도"];
fruits[4] = "복숭아"; // ['귤', '배', '포도', empty, '복숭아']

배열Array속 요소는 자료형의 제약이 없을까요?
즉, 객체{:}, function(){}이 올 수 있을까요???

YES.제약없음
let arr = [
"사과",
{ name: "이보라" },
true,
function () { console.log("안녕하세요."); },
];

뒤 빼기 배열변수.pop()

맨 앞 빼기, 추가

앞 빼기 배열변수.shift() 제일 앞 요소를 꺼내 제거한 후 남아있는 요소들을 앞으로 밀어줍니다. 이렇게 하면 두 번째 요소가 첫 번째 요소가 됩니다.
앞 추가 배열변수.unshift()
